# Maya Plugin - Lugwit Maya Plugin Suite

## 概述

这是一个全面的Maya插件套件，专注于动画制作、角色绑定、材质管理和动捕数据处理。

## 主要功能

### 🎬 动捕数据处理
- **HIK动捕系统**: 完整的HumanIK集成，支持FBX动捕数据导入和烘焙
- **T-pose处理**: 自动调整角色到标准T-pose姿势
- **骨骼映射**: 智能骨骼名称匹配和映射
- **约束系统**: 自动建立IK/FK约束关系

### 🎭 角色绑定
- **高级骨骼系统**: 支持复杂的角色绑定需求
- **控制器创建**: 自动化控制器生成和命名
- **权重工具**: 完善的权重绘制和管理工具
- **镜像系统**: 对称绑定和权重镜像

### 🎨 材质管理
- **材质同步**: 跨文件材质传递和同步
- **贴图管理**: 智能贴图路径管理和批量处理
- **渲染设置**: 一键渲染层配置和输出
- **材质库**: 标准化材质模板库

### 🔧 建模工具
- **拓扑检查**: 自动检测和修复拓扑问题
- **减面工具**: 智能减面和优化
- **UV工具**: 高级UV展开和编辑功能
- **对称工具**: 模型对称性检查和修复

## 安装说明

### 系统要求
- Maya 2018 或更高版本
- Python 3.6+
- Windows 10/11

### 安装步骤
1. 克隆或下载此仓库
2. 将整个文件夹复制到Maya插件目录
3. 在Maya中加载插件
4. 重启Maya以完成安装

## 使用指南

### 动捕数据处理
1. 在Maya中打开角色文件
2. 运行"导入并烘焙动捕数据"工具
3. 选择T-pose节点和FBX文件
4. 设置帧速率并开始处理

### 角色绑定
1. 选择角色模型
2. 使用绑定工具创建骨骼系统
3. 调整控制器位置
4. 应用权重和约束

### 材质管理
1. 选择目标物体
2. 使用材质同步工具
3. 配置材质参数
4. 批量应用到多个物体

## 文件结构

```

├── l_scripts/           # 核心Python脚本
│   └── Rig/           # 绑定相关脚本
│       ├── HIK_py3.py # HIK动捕处理主脚本
│       └── ...        # 其他绑定工具
├── plug-ins/          # Maya插件文件
├── icons/             # UI图标和资源
├── mov_help/          # 帮助视频

```
